> 一个智能聊天AI工具，支持问答、历史会话查看等多种功能。采用前后端分离架构，基于SpringBoot AI和Deepseek构建，提供流畅的解答体验的功能体系。

## 功能特性

### 核心功能
    支持了deepseek等多个接口，实现聊天解答的功能。

### 技术亮点
- **前后端分离**: Vue.js + Spring Boot全栈解决方案
- **大模型**: deepseek接入
